# inf8770-tp1

## Installation et Exécution

1. **Créer et activer un environnement virtuel** :
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```

2. **Installer les dépendances** :
   ```bash
   pip install -r requirements.txt
   ```

3. **Exécuter le programme** :
   ```bash
   python src/main.py
   ```
